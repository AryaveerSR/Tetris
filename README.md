# `Tetris`

## A Tetris clone in C and SDL(2).

[![MIT](https://img.shields.io/crates/l/bitvec.svg?style=for-the-badge)](#License)

</div>

## Building

Install `libsdl2-dev` and `libsdl2-ttf-dev`:

```bash
sudo apt install libsdl2-dev libsdl2-ttf-dev
```

Run the Makefile:

```bash
make
```
